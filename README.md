# Converting-HTML-Profile-Card-to-JSX

# User Profile JSX Conversion

## Objective
Convert standard HTML into JSX inside a React functional component, following best practices in React.

## Project Structure

```bash
profile-card/
├── src/
│   ├── App.js
│   ├── UserProfile.js
│   ├── App.css
│   └── images/
│       └── placeholder.jpg
├── public/
└── package.json
```
## Features
- Functional React component
- Uses JSX syntax
- Proper `className` usage
- Self-closing tags
- Image imported from `src/images`
- Props for dynamic user information

## How to Run

1. Install dependencies and run app:

```bash
npm install
npm start
```
